# 包

+ 包结构
  + 用于组织包中的各种文件
  + 是一个压缩文件，解压后还原为目录。
  + 符合规范的目录， 应该包含如下文件：
    + **package.json     描述文件**
    + bin                       可执行的二进制文件
    + lib                         js代码
    + doc                       文档
    + test                       单元测试
+ 包描述文件
  + 描述包的相关信息，以供外部读取分析

+ 创建包
  + npm init
+ 常用指令
  + npm install 包名 --save   设置为依赖
